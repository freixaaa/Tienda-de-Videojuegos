# tienda de videojuegos

## 1. descripcion del proyecto
este proyecto consiste en un sistema en consola desarrollado en python que permite realizar distintas acciones relacionadas con una tienda de videojuegos; el usuario puede administrar un catalogo, utilizar un carrito de compras y generar una factura al finalizar una compra.

## 2. funcionalidades que generan los codigos
1. cargar el catalogo de videojuegos desde archivos json o csv proporcionados por el profesor
2. agregar nuevos videojuegos desde la consola
3. simular un carrito de compras para seleccionar videojuegos
4. generar una factura al finalizar la compra

## 3. tecnologias
- python

## 4. ejecucion
para ejecutar el programa, se debe correr el archivo principal:
python main.py
